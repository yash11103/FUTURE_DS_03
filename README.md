This project focuses on analyzing an e-commerce user journey to identify bottlenecks in the conversion funnel. The primary objective was to perform a comprehensive funnel analysis on a dataset containing 21,000+ user sessions. The task involved:

Funnel Construction: Tracking users through sequential stages: Browse ‚Üí Add to Cart ‚Üí Checkout ‚Üí Purchase.
Conversion Metrics: Calculating drop-off rates and conversion percentages at each stage of the funnel.
Segmentation Analysis: Evaluating funnel performance across different dimensions such as Device Type, Region, and Marketing Channel (e.g., Organic, Social Media, Google Ads).
Trend & Pattern Discovery: Identifying peak activity hours and daily session trends to understand temporal user behavior.
Automated Reporting: Generating a multi-sheet Excel report (funnel_analysis_report.xlsx) for corporate stakeholders and saving high-resolution visualizations of conversion rates.
